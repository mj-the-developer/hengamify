  • injectElements(elements: HTMLElement[], wrappers: string[], prepend?: boolean, injectToParent?: boolean): boolean
    • Defined in src/utils/InjectionUtils.ts:45

    Try to append or prepend the provided elements to one of the wrappers provided in the wrappers list or their parents. The method reverse the order of elements if the elements are going to prepend to the wrapper

    Parameters

    • elements: HTMLElement[]

      List of HTML elements to inject to one of the wrappers

    • wrappers: string[]

      List of desired wrappers to inject the elements into them

    • prepend: boolean = false

      Boolean to prepend the elements instead of appending

    • injectToParent: boolean = false

      Either inject the elements to wrapper itself or its parent

    Returns boolean

    boolean which indicate if the injection were successful
